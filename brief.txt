AxiomeTech is a bilingual AI solutions website serving French Tech companies and SMBs. The site provides strategic AI automation services with clear ROI and fixed-scope pricing. It supports both English and French languages with automatic browser language detection and manual language switching. The implementation uses vanilla HTML/CSS/JS with an i18n.json file for all text content, ensuring accessibility, responsiveness, and SEO optimization. No external dependencies required.
